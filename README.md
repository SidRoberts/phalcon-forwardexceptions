Sid\Phalcon\ForwardExceptions
=============================

Forward Exceptions within a Phalcon controller to an action of your choosing.

## Installing

Install using Composer:

```json
{
    "require": {
        "sidroberts/phalcon-forwardexceptions": "dev-master"
    }
}
```
